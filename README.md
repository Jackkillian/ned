## Installation
**NOTE: This project is under heavy development.**

### Simple
Install ned with pip:
```console
pip install ned-client
```

Run ned in your terminal:
```console
ned
```

### Advanced
To test the latest build for yourself, clone and run
```console
hatch run ned
```
